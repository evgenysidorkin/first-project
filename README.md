# Руководство по Git и GitHub для начинающих

## Установка Git

### Windows

Git уже предустановлен в составе пакета Git for Windows.

### macOS
Установите Git одним из способов:

Через терминал

С помощью менеджера пакетов Homebrew:

bash
brew install git
Linux
Используйте терминал и выберите команду для вашего дистрибутива:

Ubuntu/Debian: sudo apt install git

Fedora: sudo dnf install git

CentOS: sudo yum install git

## Проверка установки

После установки проверьте версию Git:
bash
git version

### Настройка Git

Глобальная конфигурация
Настройте имя пользователя и email (данные запишутся в файл ~/.gitconfig):
bash
git config --global user.name "Ваше Имя"
git config --global user.email "ваш.email@example.com"

### Проверка настроек

Просмотрите все текущие настройки:

bash
git config --list

## Базовые команды терминала

### Навигация

### Команда	Описание

* pwd	Показать текущую директорию
* ls	Показать содержимое директории
* cd <путь>	Перейти в указанную директорию
* cat <файл>	Показать содержимое файла

## Работа с файлами и директориями

### Команда	Описание

* touch <файл>	Создать файл
* mkdir <директория>	Создать директорию
* cp <источник> <цель>	Копировать файл/директорию
* mv <источник> <цель>	Переместить/переименовать
* rm <файл>	Удалить файл
* rmdir <директория>	Удалить пустую директорию
* rm -r <директория>	Рекурсивно удалить директорию

## Полезные советы

Используйте && для выполнения нескольких команд подряд
Используйте клавиши ↑ и ↓ для навигации по истории команд
Используйте Tab для автодополнения

## Работа с Git

Инициализация репозитория
bash
git init
* Отслеживание изменений bash
* Проверить статус git status
* Добавить все файлы git add --all Или git add .
* Добавить конкретный файл git add <имя_файла>

## Создание коммита

bash
git commit -m "Описание изменений"

## Просмотр истории

bash
git log

## Работа с GitHub
### Регистрация на GitHub

Перейдите на github.com
Нажмите Sign up
Заполните необходимые поля:
Email
Пароль
Имя пользователя
Пройдите капчу
Подтвердите email через код

### Создание удалённого репозитория

Нажмите New repository

Введите имя (например, first-project)

Нажмите Create repository

### Настройка SSH-ключа

Генерация ключа
bash
ssh-keygen -t ed25519 -C "ваш.email@example.com"

### Копирование публичного ключа

macOS:

bash
pbcopy < ~/.ssh/id_ed25519.pub
Windows:

bash
clip < ~/.ssh/id_ed25519.pub
Linux:

bash
cat ~/.ssh/id_ed25519.pub

### Добавление ключа на GitHub

Перейдите в Settings → SSH and GPG keys

Нажмите New SSH key

Вставьте ключ

Нажмите Add SSH key

### Проверка подключения

bash
ssh -T git@github.com

### Связывание локального и удалённого репозиториев
Добавление удалённого репозитория
bash
git remote add origin git@github.com:username/repository.git

### Проверка подключения

bash
git remote -v
